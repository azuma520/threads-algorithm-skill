# Threads 發文 Checklist

## 發文前（必須全部通過）

### 時機檢查
- [ ] 距離上一篇至少 24 小時？
- [ ] 上一篇的 freshness 週期已結束？（互動是否已趨緩）
- [ ] 核心受眾現在在線？（根據你的經驗，幾點活躍）
- [ ] 前 3 小時你有空積極互動？

### 水溫測試
- [ ] 發文前 1-2 小時去同領域熱門貼留過言？
- [ ] 留言有被推送/有人回應？（→ Audience Affinity 還在高點）
- [ ] 如果留言反應冷 → 考慮延後發文

### 內容檢查
- [ ] 開頭自然？沒有 clickbait / AI 味？
- [ ] 內容跟帳號主軸一致？（不偏離 Creator Embedding）
- [ ] 不是上一篇爆文的高度相似延伸文？（換角度了嗎）
- [ ] 如果有外部連結——域名被分享過？跟內容相關？

## 發文後（前 3 小時黃金期）

### 互動最大化
- [ ] 每則留言都回覆了？
- [ ] 每則留言都按讚了？
- [ ] 回覆內容有引發二次互動嗎？（不只是「謝謝」）
- [ ] 有料的留言記下來，之後擴寫成獨立貼文

### 下一篇規劃
- [ ] 強制休息 24-48 小時
- [ ] 下一篇準備好了嗎？角度有切換嗎？
- [ ] 路徑規劃：爆文 → 感謝文（帶乾貨）→ 弱相關但同圈子

## 貼文沒爆時

- [ ] **不要刪文重發**（除非 10 分鐘內、零互動、有嚴重錯誤）
- [ ] 善用編輯功能（小改不影響曝光）
- [ ] 診斷原因（用 post-diagnosis 模板）
- [ ] 用不同角度重寫一篇
