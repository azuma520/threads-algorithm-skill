# 貼文沒流量？診斷流程

按順序排查，找到第一個命中的問題就是最可能的原因。

## 第一關：內容品質篩（Low Signal）

問自己：
- [ ] 開頭是不是很像 clickbait？（「你絕對想不到」「震驚」「三個原因」）
- [ ] 是不是 AI 直接生成沒有改過？
- [ ] 內容有沒有實質價值？還是只是換個方式說廢話？

→ 如果命中：內容被標記為 low signal，初始推送範圍就被壓了。
→ 解法：重寫開頭，用自然語氣，加入具體經驗或數據。

## 第二關：擴散時機

問自己：
- [ ] 距離上一篇不到 24 小時？
- [ ] 上一篇還在擴散中（還有人在留言互動）？
- [ ] 今天已經發過其他貼文？

→ 如果命中：自己跟自己對打，Diversity Enforcement 壓了你的新文。
→ 解法：等上一篇冷卻再發。下次間隔至少 24-48 小時。

## 第三關：受眾時間

問自己：
- [ ] 你的核心受眾現在在線嗎？
- [ ] 前 3 小時有沒有人互動？
- [ ] 發文後你有沒有在線回覆？

→ 如果命中：前 3 小時互動不夠，沒有觸發第二波擴散。
→ 解法：下次在受眾活躍時段發文，確保前 3 小時能積極互動。

## 第四關：帳號定位偏移

問自己：
- [ ] 這篇內容跟你的帳號主軸一致嗎？
- [ ] 最近有沒有蹭過不相關的熱門話題？
- [ ] 粉絲是因為什麼追蹤你的？跟這篇文一致嗎？

→ 如果命中：Creator Embedding 被打亂，或粉絲結構跟現在定位不 match。
→ 解法：回歸主軸，持續發同領域內容讓演算法重新學習。不要為了衝粉蹭不相關流量。

## 第五關：互動品質不足

問自己：
- [ ] 前幾則留言是「推」「太強了」還是有實質討論？
- [ ] 有沒有人私訊分享？
- [ ] 你的回覆有沒有引發二次互動？

→ 如果命中：互動信號不夠強，EV 系統判定價值不高。
→ 解法：下次在開頭或結尾加入引導討論的問題，鼓勵深度留言。

## 第六關：Audience Affinity 衰減

問自己：
- [ ] 你有多久沒發文了？
- [ ] 最近有沒有去同領域留言保持互動？
- [ ] 發文前有沒有測水溫？

→ 如果命中：跟受眾的親和力分數衰減了，初始推送效果差。
→ 解法：先去同領域熱門貼互動幾天暖場，等 Affinity 回升再發重要內容。

## 排查完都沒命中？

有時候就是運氣。演算法是幾十個參數交叉組合，不是每次都能找到單一原因。
持續產出比糾結一篇更重要。
