# 帳號權威系統

演算法怎麼看待「你是誰」？你的帳號不只是一個名字，它有語意標籤、集群歸類、權威評分。這些決定了你發的每一篇文會被推給誰。

## 核心原理

### Creator Embedding（語意標籤）

每個帳號有一組語意向量（Creator Embedding），代表演算法對你的理解：
- 你的爆文內容決定你的集群歸類（US10558714B2）
- 之後發同類型內容 → 演算法持續擴散流量
- 發偏離集群的內容 → 不給流量

這跟 Google SEO 的 topic cluster 概念一樣——持續在同一領域深耕才能建立權威。

### Social Graph 權威加權（US-9582786-B2）

不是所有互動都等值。演算法看的是**誰**跟你互動：
- 同領域的權威 KOL（追蹤比你多）留言/按讚 → 你的貼文被打高分 → 瘋狂擴散
- 關鍵：KOL 的領域標籤要跟你的內容相關才有效
- 跨領域的高追蹤 KOL 互動效果低

**對照 Google**：E-E-A-T（Experience, Expertise, Authoritativeness, Trustworthiness）同樣是帳號/網站級的權威度標籤，不是單篇文的屬性。

### 新帳號蜜月期（US20140172877A1 + US9959412B2）

新帳號有一段 exploration 測試期：
- 系統不知道你好不好，先推出去看反應
- 但內容品質還是會先過篩，垃圾一樣沉底
- 本質是 exploration vs exploitation 的權衡——給你機會收集數據，不是白送流量

## 反問句：帳號定位檢查

1. 我的帳號主軸是什麼？能用一句話說清楚嗎？
2. 最近三篇內容是否都在同一個集群內？
3. 有沒有為了蹭熱度發過跟主軸無關的文？
4. 跟我互動的 KOL 是同領域的嗎？
5. 我的粉絲是因為什麼內容追蹤我的？跟我現在發的一致嗎？

## 常見迷思破解

### 「蹭熱門話題可以快速漲粉？」
飲鴆止渴。蹭無關話題（如 WBC）可能短期高流量，但進來的受眾不會跟你後續內容互動 → 互動率下降 → Meta 判定帳號「無互動價值」→ 權重暴跌 → 整個帳號崩盤。

### 「追蹤數越高觸及越好？」
常常相反。系統推薦時看歷史互動強度（US9582786B2）。大量粉絲是靠別的內容吸來的 → 換方向後他們不互動 → 初始互動率被拉低 → 演算法不給第二輪。「粉絲少的時候觸及率反而高」不是被限流，是粉絲結構跟現在定位不 match。

### 「新帳號有偏心加成？」
不是偏心，是需要數據。系統推你出去測試，但品質不行一樣沉底。前幾篇有創意 → 直接大爆；前幾篇亂寫 → 直接冷宮，後面怎麼發都沒觸及。

## 實戰案例

- darkseoking 被歸類到「硬核 SEO」集群後，發基礎文章反而沒流量
- 北大（@north0508）轉發後再爆一次，因為北大簡介提到做過 SEO → 演算法判定為 SEO 領域權威。其他高追蹤但非 SEO 領域的 KOL 留言沒有同樣效果
- darkseoking 從 6k 衝到快兩萬，大部分是演算法內容粉（因專利系列進來），現在發 SEO 文反而冷——粉絲結構跟定位不 match 的坑正在填
